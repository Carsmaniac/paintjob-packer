# Contribution Guidelines

Paint Job Packer is open source and if you'd like to help make it better I would absolutely welcome your contribution! There are just a few rules I'd ask you to follow, please:

1. Please do not contribute code written by generative AI. Consulting AI for programming questions is okay, but AI-written code is not. "Vibe coding" is strictly prohibited.
2. Paint Job Packer does not support paid or private mods. Please only add support for mods that are free and publicly available on reputable sites, such as Steam Workshop, TruckyMods or the SCS Forums.
3. I support trans rights, equal opportunities and freedoms for people of colour and acknowledge the traditional owners of the land I work on. If you take issue with any of this, please do not contribute to this project.

**TL;DR:** No AI slop, no paid mods, no bigots
